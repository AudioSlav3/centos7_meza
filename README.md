# centos7_meza

Step 1. Log into VM as <b>root</b> and install git<br>
<code>yum install -y git</code><br>

Step 2. While still logged into VM as <b>root</b>, git clone repo<br>
<code>git clone https://github.com/AudioSlav3/centos7_meza.git /tmp</code><br>

Step 3. Run pre_init.sh
<code>cd /tmp/</code><br>
<code>bash /tmp/pre_init.sh</code><br>
